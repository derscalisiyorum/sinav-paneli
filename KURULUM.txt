RENDER GELİŞMİŞ SÜRÜM KURULUMU

1) Bu zip içindeki dosyaları GitHub repodaki mevcut dosyaların yerine yükleyin.
2) Render panelinde Manual Deploy > Deploy latest commit yapın.
3) Site açılınca admin panelinde:
   - Sınava Girenler canlı yenilenir
   - Toplam / Aktif / Biten / Ortalama puan görünür
   - CSV indir butonu ile liste dışa aktarılır
4) Bu sürüm mevcut indexiniz üzerine kurulmuştur.
